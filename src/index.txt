dottxt returns important page information from various sites as a plain text.
This can be useful in extremely low bandwidth environments.


Current URLs:

—————

/wiki/<query>
View a specific Wikipedia article, or the first search result.

—————

/wiki/search/<query>
Get a list of Wikipedia articles that match the query.

—————

/track/<carrier>/<code>
Retrieve parcel tracking information with up to 5 events.

carrier: USPS, UPS
code: valid tracking code

—————

/track/<carrier>/<code>/full
Retrieve parcel tracking information with full history.

carrier: USPS, UPS
code: valid tracking code
